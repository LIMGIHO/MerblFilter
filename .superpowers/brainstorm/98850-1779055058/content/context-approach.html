본문 / 댓글 분리 처리 방식

AI가 어떤 컨텍스트를 쓸지 어떻게 결정할까요?

A

자동 감지

질문 키워드를 분석해서 자동 선택

  • "요약", "핵심", "분석" → 본문만
  • "댓글", "반응", "사람들" → 댓글만
  • 그 외 → 본문 + 댓글
👍 사용자가 신경 쓸 필요 없음
B

컨텍스트 토글

입력창 위에 버튼으로 직접 선택

📄 본문 💬 댓글 전체
👍 명확한 제어 / 오작동 없음
C

A + B 조합

자동 감지 기본 + 토글로 수동 오버라이드

자동으로 컨텍스트 선택하되,
사용자가 원하면 직접 바꿀 수 있음
👍 가장 유연 / 복잡도 조금 높음
